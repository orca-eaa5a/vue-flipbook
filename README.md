# vue-flipbook
![Github_Logo](./demo-img/img.png)  
flipbook component of VueJS
## dependencies
```
1. jquery
2. vue-pdf
3. turn.js (http://www.turnjs.com/)
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
